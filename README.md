# 🛒 ربات جستجوی محصولات تکنولایف (Web Scraper)

![Version](https://img.shields.io/badge/version-1.0-blue)
![Python](https://img.shields.io/badge/python-3.8+-green)
![License](https://img.shields.io/badge/license-MIT-orange)

## 📋 معرفی پروژه
این یک ربات جستجوی هوشمند برای سایت **تکنولایف** است که:
- نام محصول را از کاربر می‌گیرد
- در سایت تکنولایف جستجو می‌کند
- اطلاعات محصولات (اسم، قیمت، لینک) را استخراج می‌کند
- خروجی را در یک فایل **CSV بسیار زیبا و مرتب** ذخیره می‌کند

---

## ✨ ویژگی‌ها
- ✅ جستجوی دقیق محصولات
- ✅ استخراج اسم، قیمت و لینک
- ✅ فیلتر هوشمند محصولات
- ✅ خروجی CSV با فرمت حرفه‌ای
- ✅ دارای تاریخ و ساعت در فایل خروجی
- ✅ خطوط جداکننده و ایموجی برای خوانایی بهتر

---

## 🛠️ تکنولوژی‌های استفاده شده
- **Python 3.8+**
- **BeautifulSoup4** - برای parsing HTML
- **Requests** - برای دریافت صفحه از سایت
- **CSV** - برای ذخیره خروجی
- **Datetime** - برای ثبت تاریخ و ساعت

## 📦 نصب و راه‌اندازی

### 1. کلون کردن پروژه
```bash
git clone https://github.com/your-username/technolife-scraper.git
cd technolife-scraper
```

### 2. نصب کتابخانه‌های مورد نیاز
```bash
pip install beautifulsoup4 requests
```

### 3. اجرای برنامه
```bash
python scraper.py
```

---

## 🚀 نحوه استفاده

1. برنامه را اجرا کنید
2. نام محصول مورد نظر را وارد کنید (مثال: `iphone 17` یا `سامسونگ s25`)
3. منتظر بمانید تا جستجو انجام شود
4. فایل CSV در همان پوشه ساخته می‌شود

### مثال:
```bash
📌 نام محصول را وارد کنید: iphone 17
🌐 در حال جستجوی 'iphone 17' در سایت تکنولایف...
✅ صفحه با موفقیت دریافت شد
📦 96 بخش برای بررسی پیدا شد
```

## ⚙️ تنظیمات پیشرفته

شما می‌توانید این موارد را در کد تغییر دهید:

| متغیر | توضیح | مقدار پیش‌فرض |
|-------|-------|---------------|
| `OUTPUT_FILE` | نام فایل خروجی | `نتایج_جستجو_...` |
| `HEADERS` | هدرهای درخواست | User-Agent مرورگر |
| `TIMEOUT` | زمان انتظار برای پاسخ | 10 ثانیه |

---

## ❗ خطاهای احتمالی

| خطا | دلیل | راه حل |
|-----|------|--------|
| `نام محصول نمی‌تواند خالی باشد` | ورودی خالی | یک نام معتبر وارد کنید |
| `خطا در ارتباط با سایت` | قطع اینترنت | اتصال اینترنت را بررسی کنید |
| `هیچ محصولی پیدا نشد` | محصول وجود ندارد | املای کلمه را بررسی کنید |
